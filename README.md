# Word-Dictionary-App
A clean, minimal dictionary web app thatlets you search for word definitions,examples,phonetics and synonyms in real time.

## Features.
-Search any English name.
-View definations by part of speech.
-See example sentences.
-Browse synonyms.
-LIsten to word pronounciation.

## Built with.
-HTML
-CSS
-Java Script
-[Directory API](https://dictionaryapi.dev/)

## How to use.
1. Type a word in the search bar.
2. Press the search button or hit enter.
3. View the results instantly.
